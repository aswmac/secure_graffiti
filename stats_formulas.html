College Statistics Formula Sheet

Descriptive Statistics

Mean:

μ = ∑x n

Sample Mean:

̄x = ∑x n

Variance (Population):

σ2 = ∑ (x-μ)2 n

Sample Variance:

s2 = ∑ (x-̄x)2 n-1

Probability

Addition Rule (Mutually Exclusive):

P(A∪B)=P(A)+P(B)

Multiplication Rule (Independent Events):

P(A∩B)=P(A)P(B)

Distributions

Binomial Probability:

P(X=k)= ( n! k!(n-k)! ) pk (1-p)n-k

Normal Distribution (Standardized):

z = x-μ σ

Sampling & Central Limit Theorem

Standard Error (Sample Mean):

SE = σ n

Confidence Intervals

CI for Mean (Known σ):

̄x ± z ⋅ σ n

CI for Proportion:

p ± z ⋅ p(1-p) n

Hypothesis Testing

Test Statistic (Mean, Known σ):

z = ̄x-μ_0 σ n

Regression & Correlation

Linear Regression Equation:

y = a + bx

Slope:

b = ∑ (x-̄x)(y-̄y) ∑ (x-̄x)2

Correlation Coefficient:

r = cov(x,y) s_xs_y
